Frontend (React + Vite)

Run instructions:

```powershell
cd frontend
npm install
npm run dev
```

The development server will run at `http://localhost:5173/`.

### Development Notes
- **State Management:** Zustand
- **Styling:** Tailwind CSS + Lucide Icons
- **Forms:** React Hook Form
- **API Client:** Axios (configured with interceptors in `src/api/api.ts`)

Please refer to `Project_report.pdf` for the full technical documentation.
